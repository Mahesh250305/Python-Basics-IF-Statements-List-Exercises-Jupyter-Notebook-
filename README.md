🐍 Python IF & List Exercises (Jupyter Notebook)

📌 Overview

This repository contains a collection of beginner-friendly Python exercises focused on:

- IF Statements (Decision Making)
- Functions
- Lists & String Handling

All problems are solved using clean and simple logic, making this repository ideal for beginners and students learning Python fundamentals.

---

📚 Topics Covered

🔹 IF Statement Exercises

- Even or Odd Number
- Human Years to Dog Years Conversion
- Vowel or Consonant Checker
- Shape Name from Number of Sides
- Days in a Month
- Sound Level Classification (Decibels)
- Musical Note Frequency (C0–C8)
- Frequency to Note Conversion
- Wavelength to Color (Visible Spectrum)

---

🔹 List & Function Exercises

- Remove n Smallest and n Largest Values from a List
- Extract Words from a String (Punctuation Handling)

---

🛠️ Technologies Used

- Python 3
- Jupyter Notebook

---

🚀 How to Run

1. Install Jupyter Notebook:
   
   pip install notebook

2. Open the notebook:
   
   jupyter notebook

3. Run all cells:
   
   - Click Kernel → Restart & Run All
